# Ruby

## Dependencies
- [Bundler](http://bundler.io/)

## Property-based testing framework
- [Rantly](https://github.com/hayeah/rantly)

## Setup
From this directory, run:

```
$ bundle install
```

## Run unit tests

From this directory, run:

```
$ rspec prime-factors-spec.rb
```

## Run property-based tests

From this directory, run:

```
$ rspec prime-factors-check.rb
```
