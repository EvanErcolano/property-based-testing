# Clojure

## Dependencies
- [Leiningen](http://leiningen.org/)

## Setup
From the `prime-factors` directory, run:

```
$ lein deps
```

## Run unit tests

From the `prime-factors` directory, run:

```
$ lein test prime-factors.core-test
```

## Run property-based tests

From the `prime-factors` directory, run:

```
$ lein test prime-factors.core-check
```
